# Cloned Repositories

This directory contains repositories relevant to the investigation of induction heads and their role in leakage/repetition.

1.  **TransformerLens** (github.com/neelnanda-io/TransformerLens)
    -   *Purpose*: Essential library for mechanistic interpretability of transformers.
    -   *Location*: `code/TransformerLens/`
    -   *Key functionality*: Causal interventions, activation patching, attention visualization.

2.  **Generalized Induction-Head (GIM)** (github.com/ejkim47/generalized-induction-head)
    -   *Purpose*: Implementation of the GIM paper's retrieval-based next-token prediction.
    -   *Location*: `code/GIM/`
    -   *Key functionality*: Retrieval-based module for in-context information.

3.  **Evolution of Statistical Induction Heads** (github.com/EzraEdelman/Evolution-of-Statistical-Induction-Heads)
    -   *Purpose*: Code for the paper "The Evolution of Statistical Induction Heads".
    -   *Location*: `code/Evolution-of-Statistical-Induction-Heads/`
    -   *Key functionality*: Data generation for Markov Chain ICL tasks.

4.  **MarkovICL** (github.com/Simon-Lepage/MarkovICL)
    -   *Purpose*: Alternative implementation for Markov Chain ICL.
    -   *Location*: `code/MarkovICL/`
    -   *Key functionality*: Markov Chain simulation and ICL evaluation.

5.  **Surgeon** (github.com/gussand/surgeon)
    -   *Purpose*: Code for "Even Heads Fix Odd Errors" paper.
    -   *Location*: `code/surgeon/`
    -   *Key functionality*: Head-level intervention and "repair" techniques.
